Au cours de notre première nuit de l’info, notre groupe (le Front Populaire) composé de Jean Louis Cheng, Nils Chol, Roman Didelet, Nassim El M’Rabti, Marine Elisabeth, Nicolas Jatob, Onur Can Kadioglu, Jason Khaou, Karina Moussaoui, Stella Thammavong et Aurélien Andrieux, souhaitait s’attaquer à différents défis.

Premièrement, le défi principal de la nuit de l’info. Nous avons pu créer une application à deux onglets qui récupère une base de données, et qui propose à l’utilisateur de faire un retour sur son expérience.
Nous avons aussi créé un site web sur lequel vous retrouverez la présentation de l’application Surfrider en anglais.

Vous pourrez trouver sur le lien github suivant les fichiers du projet ainsi que le fichier app-debug.apk final de l’application : https://github.com/BilboBaguette/Surfrider.

Sur la machine OVH, le dossier de projet se situe dans le home de l'utilisateur user.
